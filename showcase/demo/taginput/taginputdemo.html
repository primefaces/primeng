TagInput TagInput is used to enter tags.

Basic - {{tags}}

EMail regex - {{emails}}

Import


import {TagInput} from 'primeng/primeng';

Getting Started

Two-way binding to a Array<string> is defined using the standard ngModel directive.


<p-taginput [(ngModel)]="tags"></p-taginput>



export class MyModel {

    tags: Array<string> = [];

}

Customization

Placeholder, delimiter, regex pattern and more can be changed by properties.


<p-taginput placeholder="Add an email" [allowedTagsPattern]="validEmailPattern" delimiterCode="188" [(ngModel)]="settings.recipients"></p-taginput>

Attributes

Name Type Default Description
placeholder string 'Add a tag' Placeholder for the <input> tag.
delimiterCode string '188' ASCII keycode to split tags on. Defaults to comma.
addOnBlur boolean true Whether to attempt to add a tag when the input loses focus.
addOnEnter boolean true Whether to attempt to add a tag when the user presses enter.
addOnPaste boolean true Whether to attempt to add a tags when the user pastes their clipboard contents.
allowedTagsPattern RegExp /.+/ RegExp that must match for a tag to be added.

Dependencies

Native component that requires the css of PrimeUI Switch.

  }
