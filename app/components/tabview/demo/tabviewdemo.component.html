Getting Started

Tabview element consists of one or more p-tabPanel elements. Header of the tab is defined using header attribute.

<p-tabView>
    <p-tabPanel header="Header 1">
       Content 1
    </p-tabPanel>
    <p-tabPanel header="Header 2">
        Content 2
    </p-tabPanel>
    <p-tabPanel header="Header 3">
        Content 3    
    </p-tabPanel>
</p-tabView>

ActiveIndex

ActiveIndex property supports two-way binding, changes to the model changes the selected tab and changing the tab with user interaction updates the value at model.

<p-tabView [(activeIndex)]="index">
    <p-tabPanel header="Header 1">
       Content 1
    </p-tabPanel>
    <p-tabPanel header="Header 2">
        Content 2
    </p-tabPanel>
    <p-tabPanel header="Header 3">
        Content 3    
    </p-tabPanel>
</p-tabView>

export class ModelComponent {
    index: number: 0;
}

Disabled Tabs

TabPanel component has a disabled property, when specified tab header is not clickable.

<p-tabPanel header="Header Text" disabled="disabled">
    Content 1
</p-tabPanel>

Closable Tabs

When closable attribute is enabled, an icon is displayed to close the tab. onClose event is fired when a tab is closed.

<p-tabPanel header="Header Text" closable="closable">
    Content 1
</p-tabPanel>

Effects

An animation can be used when switching between tabs, valid values are "blind","bounce","clip","drop", "explode","fade","fold","highlight","puff","pulsate","scale","shake","size","slide","transfer". Duration of the effect is customized with effectDuration attributes using number values as milliseconds or pre-defined values such as "slow","normal" and "fast".

<p-tabView effect="fade" effectDuration="fast">

Orientation

TabView has four different orientations; "top", "bottom", "left" and "right". Top is the default one and this can be changed using orientation attribute.

<p-tabView orientation="left">

Attributes

Name Type Default Description
activeIndex number 0 Index of the active tab.
orientation String top Orientation of tab headers.
effect String null Animation to use when switching tabs.
effectDuration any null Duration of the effect in milliseconds or pre-defined values like "slow","normal" and "fast".
onChange function null Callback to invoke on tab change.
onClose function null Callback to invoke on tab close.

Events

Name Parameters Description
onChange originalEvent: Browser event
index: Index of the selected tab 		Callback to invoke on tab change.
onClose originalEvent: Browser event
index: Index of the closed tab 		Callback to invoke on tab close. 
<p-tabView (onChange)="handleChange($event)">

export class ModelComponent {
    
    handleChange(e) {
        var index = e.index;
    }
}
