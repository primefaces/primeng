Animate

Animate manages PrimeFlex CSS classes declaratively to during enter/leave animations on scroll or on page load.

Import
import {AnimateModule} from 'primeng/animate';
Getting Started

Animate uses PrimeFlex animations, however it can perform animations with custom CSS classes too. Takes two properties enterClass and leaveClass to simply add-remove a class and enter/leave during scroll or page load to manage elements animation if the element is in viewport.
