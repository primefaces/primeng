Getting Started

Two-way value binding is defined using the standard ngModel directive referencing to either a JavaScript Date, a DateTime Object (luxon) or a plain numeric timestamp property.
To specify the preferred property type for the date element just set the dataType property to either 'date', 'luxon' or 'timestamp' (the default is 'date'). 


<p-datepicker [(ngModel)]="value" [dataType]="dataType"></p-datepicker>



export class MyModel {

    dataType: 'date' | 'luxon' | 'timestamp';
    value: Date | DateTime | number;

}

Model Driven Forms

Calendar can be used in a model driven form as well.


<p-datepicker formControlName="date"></p-datepicker>

Popup and Inline

Calendar is displayed in a popup by default and inline property needs to be enabled for inline mode.


<p-datepicker [(ngModel)]="value" [inline]="true"></p-datepicker>

Selection Mode

By default calendar allows selecting one date and multiple dates can be selected by setting selectionMode to multiple. In this case calendar updates the value with an array of dates where optionally number of selectable dates can be restricted with maxDateCount property. Third alternative is the range mode that allows selecting a range based on an array of two values where first value is the start date and second value is the end date.

DateFormat

Default date format is dd/MM/yyyy, to customize this use dateFormat property.


<p-datepicker [(ngModel)]="dateValue" dateFormat="dd.MM.yy"></p-datepicker>

Following options can be a part of the format.

  • d - day of month (no leading zero)
  • dd - day of month (two digit)
  • M - month of year (no leading zero)
  • MM - month of year (two digit)
  • y - year (no leading zeros)
  • yy - year (two digit)
  • yyyy - year (four digit)
  • a - meridian indicator (AM | PM)
  • anything else - literal text

Time

TimePicker is enabled increasing picker precision by setting it either to 'hour', 'minute' or 'second' property and 24 (default) or 12 hour mode is configured using hourFormat option. 


<p-datepicker [(ngModel)]="value" precision="minute" hourFormat="12"></p-datepicker>
<p-datepicker [(ngModel)]="value" precision="second" hourFormat="24"></p-datepicker>

Date Restriction

To disable entering dates manually, set readonlyInput to true and to restrict selectable dates use minDate and maxDate options.
Note: minDate and maxDate must be of the same type specified with the dataType attribute. 


<p-datepicker [(ngModel)]="dateValue" [minDate]="minDateValue" [maxDate]="maxDateValue" readonlyInput="true"></p-datepicker>

Disable specific dates and/or days

To disable specific dates or days, set readonlyInput to true and to restrict selectable dates use disabledDates and/or disabledDays options.
Note: disabledDates and disabledDays must be of the same type specified with the dataType attribute. 


<p-datepicker [(ngModel)]="dateValue" [disabledDates]="invalidDates" [disabledDays]="[5,6]" [readonlyInput]="true"></p-datepicker>

Button Bar

Button bar displays today and clear buttons and enabled using showButtonBar property.


<p-datepicker [(ngModel)]="dateValue" [showButtonBar]="true"></p-datepicker>

Localization

Localization for different languages and formats is defined by binding the locale settings object to both the 'locale' and 'firstDayOfWeek' properties. Following is the default values for English.
Note: firstDayOfWeek defaults to Monday (the values are 0-6 Monday-Sunday) 


<p-datepicker [(ngModel)]="dateValue" [locale]="en" [firstDayOfWeek]="6"></p-datepicker>

Custom Content

Calendar UI accepts custom content using p-header and p-footer components.


<p-datepicker [(ngModel)]="dateValue">
    <p-header>Header</p-header>
    <p-footer>Footer</p-footer>
</p-datepicker>

In addition, cell contents can be templated using an ng-template with a pTemplate directive whose value is either "year", "month", "date", "hour", "minute" or "second". This is a handy feature to highlight specific elements.
Note that the implicit variable passed to the template is not a date instance but a metadata object to represent that element. The available properties are 'year', 'month', 'day', 'hour', 'minute', 'second' properties.
Example below changes the background color of dates between 10th and 21st of each month. 


<p-datepicker [(ngModel)]="date10">
    <ng-template pTemplate="date" let-date>
        <span style="border-radius:10%;"
              [style.text-align]="(date.day < 21 && date.day > 10) ? 'center' : null"
              [style.background-color]="(date.day < 21 && date.day > 10) ? '#7cc67c' : null">
            {{date.day}}
        </span>
    </ng-template>
</p-datepicker>

            dayNamesMin: [ "L","M","X","J","V","S","D" ],
            monthNames: [ "enero","febrero","marzo","abril","mayo","junio","julio","agosto","septiembre","octubre","noviembre","diciembre" ],
            monthNamesShort: [ "ene","feb","mar","abr","may","jun","jul","ago","sep","oct","nov","dic" ],
            today: 'Hoy',
            clear: 'Borrar',
            pm: 'PM',
            am: 'AM'
        }

        this.tr = {
            firstDayOfWeek : 1
        }

        let today = new Date();
        let month = today.getMonth();
        let year = today.getFullYear();
        let prevMonth = (month === 0) ? 11 : month -1;
        let prevYear = (prevMonth === 11) ? year - 1 : year;
        let nextMonth = (month === 11) ? 0 : month + 1;
        let nextYear = (nextMonth === 0) ? year + 1 : year;
        this.minDate = new Date();
        this.minDate.setMonth(prevMonth);
        this.minDate.setFullYear(prevYear);
        this.maxDate = new Date();
        this.maxDate.setMonth(nextMonth);
        this.maxDate.setFullYear(nextYear);

        let invalidDate = new Date();
        invalidDate.setDate(today.getDate() - 1);
        this.invalidDates = [today,invalidDate];
    }
}
