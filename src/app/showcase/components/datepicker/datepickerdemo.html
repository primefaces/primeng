Getting Started

Two-way value binding is defined using the standard ngModel directive referencing to either a JavaScript Date, a DateTime Object (luxon) or a plain numeric timestamp property.
To specify the preferred property type for the date element just set the dataType property to either 'date', 'luxon' or 'timestamp' (the default is 'date'). 


<p-datepicker [(ngModel)]="value" [dataType]="dataType"></p-datepicker>



export class MyModel {

    dataType: 'date' | 'luxon' | 'timestamp';
    value: Date | DateTime | number;

}

Model Driven Forms

Calendar can be used in a model driven form as well.


<p-datepicker formControlName="date"></p-datepicker>

Popup and Inline

Calendar is displayed in a popup by default and inline property needs to be enabled for inline mode.


<p-datepicker [(ngModel)]="value" [inline]="true"></p-datepicker>

Selection Mode

By default calendar allows selecting one date and multiple dates can be selected by setting selectionMode to multiple. In this case calendar updates the value with an array of dates where optionally number of selectable dates can be restricted with maxDateCount property. Third alternative is the range mode that allows selecting a range based on an array of two values where first value is the start date and second value is the end date.

DateFormat

Default date format is dd/MM/yyyy, to customize this use dateFormat property.


<p-datepicker [(ngModel)]="dateValue" dateFormat="dd.MM.yy"></p-datepicker>

Following options can be a part of the format.

  • d - day of month (no leading zero)
  • dd - day of month (two digit)
  • M - month of year (no leading zero)
  • MM - month of year (two digit)
  • y - year (no leading zeros)
  • yy - year (two digit)
  • yyyy - year (four digit)
  • a - meridian indicator (AM | PM)
  • anything else - literal text

Time

TimePicker is enabled increasing picker precision by setting it either to 'hour', 'minute' or 'second' property and 24 (default) or 12 hour mode is configured using hourFormat option. 


<p-datepicker [(ngModel)]="value" precision="minute" hourFormat="12"></p-datepicker>
<p-datepicker [(ngModel)]="value" precision="second" hourFormat="24"></p-datepicker>

Date Restriction

To disable entering dates manually, set readonlyInput to true and to restrict selectable dates use minDate and maxDate options.
Note: minDate and maxDate must be of the same type specified with the dataType attribute. 


<p-datepicker [(ngModel)]="dateValue" [minDate]="minDateValue" [maxDate]="maxDateValue" readonlyInput="true"></p-datepicker>

Disable specific dates and/or days

To disable specific dates or days, set readonlyInput to true and to restrict selectable dates use disabledDates and/or disabledDays options.
Note: disabledDates and disabledDays must be of the same type specified with the dataType attribute. 


<p-datepicker [(ngModel)]="dateValue" [disabledDates]="invalidDates" [disabledDays]="[5,6]" [readonlyInput]="true"></p-datepicker>

Button Bar

Button bar displays today and clear buttons and enabled using showButtonBar property.


<p-datepicker [(ngModel)]="dateValue" [showButtonBar]="true"></p-datepicker>

Localization

Localization for different languages and formats is defined by binding the locale settings object to both the 'locale' and 'firstDayOfWeek' properties. Following is the default values for English.
Note: firstDayOfWeek defaults to Monday (the values are 0-6 Monday-Sunday) 


<p-datepicker [(ngModel)]="dateValue" [locale]="en" [firstDayOfWeek]="6"></p-datepicker>

Custom Content

Calendar UI accepts custom content using p-header and p-footer components.


<p-datepicker [(ngModel)]="dateValue">
    <p-header>Header</p-header>
    <p-footer>Footer</p-footer>
</p-datepicker>

In addition, cell contents can be templated using an ng-template with a pTemplate directive whose value is either "year", "month", "date", "hour", "minute" or "second". This is a handy feature to highlight specific elements.
Note that the implicit variable passed to the template is not a date instance but a metadata object to represent that element. The available properties are 'year', 'month', 'day', 'hour', 'minute', 'second' properties.
Example below changes the background color of dates between 10th and 21st of each month. 


<p-datepicker [(ngModel)]="date10">
    <ng-template pTemplate="date" let-date>
        <span style="border-radius:10%;"
              [style.text-align]="(date.day < 21 && date.day > 10) ? 'center' : null"
              [style.background-color]="(date.day < 21 && date.day > 10) ? '#7cc67c' : null">
            {{date.day}}
        </span>
    </ng-template>
</p-datepicker>

Properties

Name Type Default Description
zone string local Set timezone for the current instance of date picker.
Valid values are 'utc', 'local', any string like 'utc+#' or 'utc-#' example 'utc+3'.
dataType string date Set the data-type to be used in this instance of Datepicker valid values are 'date', 'luxon' or 'timestamp'.
defaultDate Date | DateTime | number null Set the date to highlight on first opening if the field is blank (types according to specified data-type).
selectionMode string single Defines the quantity of the selection, valid values are 'single', 'multiple' and 'range'.
style string null Inline style of the component.
styleClass string null Style class of the component.
inputStyle string null Inline style of the input field.
inputStyleClass string null Style class of the input field.
placeholder string null Placeholder text for the input.
disabled boolean false When specified, disables the component.
dateFormat string dd/MM/yyyy Format of the date.
inline boolean false When enabled, displays the calendar as inline. Default is false for popup mode.
showOtherMonths boolean true Whether to display dates in other months (non-selectable) at the start or end of the current month.
showIcon boolean false When enabled, displays a button with icon next to input.
showOnFocus boolean true When disabled, datepicker will not be visible with input focus.
icon string pi pi-calendar Icon of the calendar button.
appendTo any null Target element to attach the overlay, valid values are "body" or a local ng-template variable of another element.
readonlyInput boolean null When specified, prevents entering the date manually with keyboard.
minDate Date | DateTime | number null The minimum selectable date (types according to specified data-type).
maxDate Date | DateTime | number null The maximum selectable date (types according to specified data-type).
disabledDates Array<Date | DateTime | number> null Array with dates that should be disabled (not selectable, types according to specified data-type).
disabledDays Array<number> null Array with weekday numbers that should be disabled (not selectable).
monthNavigator boolean true Whether the month should be clickable to allow navigation.
yearNavigator boolean false Whether the year should be clickable to allow navigation.
precision string day Set the precision of the date picker, valid values are 'year', 'month', 'day', 'hour', 'minute', 'second'.
hourFormat string 24 Specifies 12 or 24 hour format.
locale object (DatePickerLocaleData interface) null An object having regional configuration properties for the calendar.
firstDayOfWeek number 0 (Monday) Specifies the first day of the month valid values are 0-6 representing Monday-Sunday.
timeOnly boolean false Whether to display timepicker only.
required boolean false When present, it specifies that an input field must be filled out before submitting the form.
tabindex string null Index of the element in tabbing order.
maxDateCount number null Maximum number of selectable dates in multiple mode.
showButtonBar boolean false Whether to display today and clear buttons at the footer
todayButtonStyleClass string ui-secondary-button Style class of the today button.
clearButtonStyleClass string ui-secondary-button Style class of the clear button.
baseZIndex number 0 Base zIndex value to use in layering.
autoZIndex boolean true Whether to automatically manage layering.
panelStyleClass string null Style class of the datetimepicker container element.
keepInvalid boolean false Keep invalid value when input blur.
hideOnDateTimeSelect boolean false Whether to hide the overlay on date selection when showTime is enabled.

Events

Name Parameters Description
onFocus event: Focus event Callback to invoke on focus of input field.
onBlur event: Blur event Callback to invoke on blur of input field.
onClose event: Close event Callback to invoke when datepicker panel is closed.
onInput event: Input event Callback to invoke when input field is being typed.
onTodayClick event: Click event Callback to invoke when today button is clicked.
onClearClick event: Click event Callback to invoke when clear button is clicked.

export class DatepickerDemo {
    date1: Date;

    date2: Date;

    date3: Date;

    date4: Date;

    date5: Date;

    date6: Date;

    date7: Date;

    date8: Date;

    date9: Date;

    date10: Date;

    date11: Date;

    dates: Date[];

    rangeDates: Date[];

    minDate: Date;

    maxDate: Date;

    invalidDates: Array<Date>;

    public esLocale: DatePickerLocaleData;

    ngOnInit() {
        this.esLocale = {
            dayNames: ['lunes', 'martes', 'miércoles', 'jueves', 'viernes', 'sábado', 'domingo'],
            dayNamesShort: ['lun', 'mar', 'mié', 'jue', 'vie', 'sáb', 'dom'],
            dayNamesMin: ['L', 'M', 'X', 'J', 'V', 'S', 'D'],
            monthNames: ['enero', 'febrero', 'marzo', 'abril', 'mayo', 'junio', 'julio', 'agosto', 'septiembre', 'octubre', 'noviembre', 'diciembre'],
            monthNamesShort: ['ene', 'feb', 'mar', 'abr', 'may', 'jun', 'jul', 'ago', 'sep', 'oct', 'nov', 'dic'],
            pm: 'PM',
            am: 'AM',
            today: 'Hoy',
            clear: 'Borrar',
            year: 'Año',
            month: 'Mes',
            hour: 'Hora',
            minute: 'Minuto',
            second: 'Segundo',
            fromHour: 'Desde la Hora',
            fromMinute: 'Desde el Minuto',
            fromSecond: 'Desde el segundo',
            toHour: 'A la Hora',
            toMinute: 'Al Minuto',
            toSecond: 'Al Segundo'
        };

        let today = new Date();
        let month = today.getMonth();
        let year = today.getFullYear();
        let prevMonth = (month === 0) ? 11 : month - 1;
        let prevYear = (prevMonth === 11) ? year - 1 : year;
        let nextMonth = (month === 11) ? 0 : month + 1;
        let nextYear = (nextMonth === 0) ? year + 1 : year;
        this.minDate = new Date();
        this.minDate.setMonth(prevMonth);
        this.minDate.setFullYear(prevYear);
        this.maxDate = new Date();
        this.maxDate.setMonth(nextMonth);
        this.maxDate.setFullYear(nextYear);

        let invalidDate = new Date();
        invalidDate.setDate(today.getDate() + 3);
        this.invalidDates = [today, invalidDate];
    }
}
