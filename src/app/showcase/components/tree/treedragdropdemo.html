Tree DragDrop

Nodes can be reordered within the same tree and also can be transferred between other trees using drag&drop.

Single Tree

Node reordering within a tree.

Multiple Trees

Node reordering between multiple tree based on scope constraints.

Files

Can transfer to Server 2.

Server 1

Can transfer to Files.

Server 2

Can transfer to Server 1.
