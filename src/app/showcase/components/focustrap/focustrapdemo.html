Focus Trap Focus Trap keeps focus within a certain DOM element while tabbing.

Input

Float Label

Disabled Input

Input with tabindex -1

Button

Disabled Button

Button with tabindex -1

Dropdown

Editor

Import


import {FocusTrapModule} from 'primeng/focustrap';

Getting Started

FocusTrap is applied to a container element with the pFocusTrap directive.


<div pFocusTrap>
    //content
</div>

Properties

Directive has no properties.

Events

Directive has no events.

Styling

Directive does not apply any styling to host.

Dependencies

None.

<div pFocusTrap>
    <h3 class="first">Input</h3>

    <h3>Disabled Input</h3>

    <h3>Input with tabindex -1</h3>
    <input type="text" size="30" pInputText tabindex="-1" />

    <h3>Button</h3>

    <h3>Disabled Button</h3>

    <h3>Button with tabindex -1</h3>
    <button pButton type="button" icon="pi pi-check" tabindex="-1"  label="Check"></button>
    
    <h3>Dropdown</h3>

            </span>
        </p-header>
    </p-editor>
</div>
