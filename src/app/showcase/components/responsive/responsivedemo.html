Responsive .ui-fluid style class provides fluid width to components for efficient use of space in screen. This example demonstrates ui-fluid in combination with Grid CSS and the components having built-in responsive modes like Table. Note that Grid CSS is not mandatory, any grid system can be used with ui-fluid.
